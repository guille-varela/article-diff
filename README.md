# Article Diff

Prototipo de control de versiones y ramas para la edición de artículos.
Son páginas HTML autónomas (sin build, sin dependencias): basta abrirlas en el navegador.

## Vistas

- **`editor-versiones.html`** — Editor de artículos con versiones y ramas.
- **`visor-versiones.html`** — Visor para comparar y revisar el historial de versiones.
- **`prd.html`** — Documentación del producto (PRD) renderizada para el navegador.
- **`PRD-editor-versiones.md`** — Fuente Markdown del PRD.
- **`index.html`** — Portada con enlaces a las vistas y a la documentación.

## Cómo verlo

Tienes dos opciones:

1. **Online (GitHub Pages):** abre el enlace publicado del repo en el navegador.
2. **Local:** descarga el repo y abre `index.html` (o cualquiera de los HTML) directamente en tu navegador.
